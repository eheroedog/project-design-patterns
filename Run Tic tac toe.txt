
Use this command to run the project
java -cp foo.jar full.package.name.ClassName

running project:
java -cp test.jar com.fproject.main.Main

running tic base game:
java -cp tic.jar com.github.robertbraeutigam.tictactoe.main.Main

Or alternatively add a MANIFEST File to the project
